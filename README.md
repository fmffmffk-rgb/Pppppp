# موقع مشاركة النصوص - TextShare

موقع تفاعلي لمشاركة النصوص مع نظام مستخدمين متكامل وإدارة للمحتوى.

## المميزات

- إنشاء حساب جديد للمستخدمين
- تسجيل الدخول للمستخدمين الحاليين
- نشر منشورات نصية من قبل المستخدمين المسجلين
- عرض جميع المنشورات بشكل عام
- لوحة تحكم للمدير (Admin) لإدارة المستخدمين والمنشورات

## التثبيت والتشغيل محلياً

### المتطلبات
- Node.js (الإصدار 14 أو أحدث)
- PostgreSQL (للبيانات)
